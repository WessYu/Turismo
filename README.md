## Sobre o projeto

Este projeto simula uma página institucional para apresentação de destinos e experiências. O foco foi construir uma interface organizada, visualmente agradável e adaptada para diferentes tamanhos de tela.

---

## Funcionalidades

- Hero section com chamada principal
- Seções informativas
- Organização de conteúdo visual
- Layout responsivo
- Estrutura simples e objetiva
- Código organizado para estudo e portfólio
